# Copes' Islander

Marketing website for Copes' Islander oceanfront vacation rental in Comox Valley, BC.

## Status

V1 in development. Inquiry-only (no booking/payments). Tech stack: TBD (Astro + Tailwind planned).

## Local development

A `dev-assets/` directory is gitignored for local-only files (photo references, scratch work, etc.). Create it locally if it doesn't exist:
mkdir -p dev-assets
